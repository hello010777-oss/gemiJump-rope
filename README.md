# 3단계 도전 줄넘기

어린이가 줄넘기 동작을 3단계로 연습할 수 있는 브라우저 기반 React 앱입니다.

## Privacy

- 카메라 권한은 사용자가 시작 버튼을 누른 뒤에만 요청합니다.
- 카메라 영상은 브라우저 안에서만 표시되고 서버로 전송되지 않습니다.
- 로그인, 결제, 위치 정보, 마이크, 외부 AI API를 사용하지 않습니다.
- 기념 사진은 사용자가 다운로드 버튼을 누른 경우에만 기기에 저장됩니다.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
